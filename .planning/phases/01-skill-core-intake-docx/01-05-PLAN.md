Create `~/.claude/skills/pbi-docgen/scripts/generate.py` with:

**Imports:**
```python
import sys
import os
import json
import time
```

**Function: `check_dependencies() -> bool`**
- Per SCAF-04. Try importing: docx, jinja2, yaml
- For each failed import, print to stderr: f"Missing dependency: {name}. Install with: pip install {package}"
- Map: docx -> "python-docx==1.2.0", jinja2 -> "Jinja2", yaml -> "PyYAML"
- Returns True if all present, False if any missing

**Function: `main(config_path: str) -> None`**
- Per D-21: single entry point with pipeline stages.
- Pipeline:

1. **Check dependencies:**
   ```python
   if not check_dependencies():
       sys.exit(1)
   ```

2. **Load config:**
   ```python
   # Add skill root to sys.path for imports
   skill_root = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
   if skill_root not in sys.path:
       sys.path.insert(0, skill_root)

   from scripts.utils import load_json_config, ensure_directory, validate_file_exists
   from scripts.md_parser import parse_markdown_sources
   from scripts.content_generator import generate_all_sections
   from scripts.docx_builder import build_docx

   config = load_json_config(config_path)
   ```

3. **Validate file paths:**
   - validate_file_exists for each file in config["source_files"]
   - validate_file_exists for config["client_logo"]
   - validate_file_exists for config["company_logo"]
   - Print to stderr: "All input files validated."

4. **Ensure output directory:**
   ```python
   ensure_directory(config["output_dir"])
   ```

5. **Parse Markdown (per D-21 stage 1):**
   ```python
   print("Stage 1/3: Parsing source Markdown...", file=sys.stderr)
   parsed = parse_markdown_sources(config["source_files"])
   print(f"  Found {len(parsed)} sections: {', '.join(parsed.keys())}", file=sys.stderr)
   ```
   If parsed is empty, print warning: "No sections detected in source files. Check that source Markdown contains ## headings matching known section categories." and sys.exit(1).

6. **Generate content (per D-21 stage 2):**
   ```python
   print("Stage 2/3: Generating section content...", file=sys.stderr)
   start = time.time()
   sections = generate_all_sections(
       parsed,
       config["client_name"],
       config["report_name"],
       config["audience"]
   )
   elapsed = time.time() - start
   print(f"  Content generated in {elapsed:.1f}s", file=sys.stderr)
   ```

7. **Build DOCX (per D-21 stage 3):**
   ```python
   print("Stage 3/3: Building Word document...", file=sys.stderr)
   output_path = build_docx(sections, config)
   file_size = os.path.getsize(output_path)
   print(f"\nComplete! Output: {output_path} ({file_size:,} bytes)", file=sys.stderr)
   ```

8. **Print output path to stdout** (for SKILL.md to capture):
   ```python
   print(output_path)
   ```

**Entry point:**
```python
if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Usage: python generate.py <config.json>", file=sys.stderr)
        sys.exit(1)
    main(sys.argv[1])
```

All progress messages go to stderr. Only the final output path goes to stdout. This lets SKILL.md capture the path cleanly.
